# Images requises

Place tes images dans ce dossier avec les noms suivants :

- `hero-bg.jpg` - Image de fond pour la section hero (page d'accueil)
- `training-bg.jpg` - Image de fond pour la section entrainement
- `cta-bg.jpg` - Image de fond pour la section contact/CTA
- `logo-grind.png` - Logo du site

Ces fichiers seront automatiquement copiés dans le dossier `dist/images/` lors du build.
